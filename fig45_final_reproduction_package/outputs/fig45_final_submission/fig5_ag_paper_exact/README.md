# Figure 5A/G Paper-Style Reproduction

This folder contains focused Figure 5A/G panels in the same two-row bar-profile style as the Kang24 paper.

Files:

```text
fig5_ag_paper_exact.svg
fig5_ag_paper_exact.png
fig5_ag_paper_exact_4x.svg
fig5_ag_paper_exact_4x.png
fig5_ag_paper_exact_metrics.csv
make_fig5_ag_paper_exact.py
```

Data used:

```text
Panel A: A-substitution profile
Panel G: T-substitution profile
Top row of each panel: MPRA observed Δactivity
Bottom row of each panel: model-predicted Δactivity
```

The visual style follows the paper: blue observed bars, red model bars, CRE/cryptic shaded regions, mutation labels, position axis, and model-panel R/R²/RMS annotations.
