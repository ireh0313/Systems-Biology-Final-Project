# Figure 5B/H Paper-Style Reproduction

This folder contains focused Figure 5B/H WT TFBS occupancy maps in the same visual style as the Kang24 paper.

Files:

```text
fig5_bh_paper_exact.svg
fig5_bh_paper_exact.png
fig5_bh_paper_exact_4x.svg
fig5_bh_paper_exact_4x.png
fig5_bh_paper_exact_metrics.csv
fig5_bh_wt_tfbs_occupancy_sites.csv
make_fig5_bh_paper_exact.py
```

Data used:

```text
WT gene: synCRE_Promega_0
Model: corrected 4TF_self seed0
TF rows: ATF7, ATF1, CREM, CREB1
Box transparency: calculated fractional occupancy
```

The visual style follows the paper: TFBS boxes with colored outlines, transparency for fractional occupancy, CRE/cryptic region marks along the bottom, and a High F / Low F legend.
